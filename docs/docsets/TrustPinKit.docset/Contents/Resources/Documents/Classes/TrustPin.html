TrustPin

public final class TrustPin : @unchecked Sendable

SSL certificate pinning for iOS, macOS, tvOS, watchOS, and visionOS.

Use the static API for an app that only needs one pinning context:

try await TrustPin.setup(TrustPinConfiguration(
    organizationId: "org", projectId: "proj", publicKey: key
))
try await TrustPin.verify(domain: "api.example.com", certificate: pem)

Libraries and multi-tenant apps should create an isolated instance via instance(id:) so they never interfere with the host’s configuration.

All operations are thread-safe.

Instance API

  • setup(_:) Asynchronous

    Configures this instance with the given credentials.

    Setup is idempotent on the success path: once an instance is configured, further setup calls return silently without re-applying credentials. To reconfigure with different credentials, create a new named instance via instance(id:) instead.

    Throws

    • invalidProjectConfig if any credential is missing, malformed, or empty after trimming, or if another setup call is already in flight on this instance.
    • errorFetchingPinningInfo if the pinning payload cannot be retrieved.
    • configurationValidationFailed if the payload signature does not validate against publicKey.

  • fetchCertificate(host:port:) Asynchronous

    Returns the server’s leaf TLS certificate as a PEM string.

    host must be a bare hostname — URL-syntax characters (/, ?, #, @) are rejected up front to prevent a malformed value from silently retargeting the connection. Pass a value like "api.example.com", not a full URL.

    Throws

    invalidServerCert if host is invalid, port is out of range, or the TLS handshake / OS trust evaluation fails.

    Declaration

    Swift

    public func fetchCertificate(host: String, port: Int = 443) async throws -> String

    Parameters

    host

    Hostname to connect to. Must be RFC-1123 valid; URL-syntax characters are not permitted.
    port

    TCP port. Must be in 1...65535. Defaults to 443.

    Return Value

    The PEM-encoded leaf certificate.
